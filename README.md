# MR-3 Virtual Controller Firmware

This repository contains the firmware binary for the **MR-3 Mini Virtual Controller**.

The firmware is intended to be flashed onto an ESP32 and used exclusively with the official **MR-3 Mini ROS 2 software stack** to evaluate and interact with the MR-3 Mini virtual robot.

## Installation

The recommended installation method is through the official Modena Robotics website:

**https://modena-robotics.com/mr3-mini-virtual-controller**
